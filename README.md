### PETNexus
* 1인 가구가 늘어나는 요즘 세상. 반려동물을 위한 커뮤니티  ”PetNexus”를 소개합니다.
* Nexus : 연결, 연계, 관계, 집합체, 무언가의 중심을 뜻하는 영어 단어로써
* 반려동물 관련 정보 및 커뮤니티를 연결하는 역할을 하기위해 만들어 졌습니다.

### 핵심기능
1. 오픈채팅 기능
2. 중고거래 기능
3. 반려동물 병원 및 shop 검색
4. 소통을 위한 커뮤니티
